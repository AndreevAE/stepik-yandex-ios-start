Test - конфигурация для тестирования приложения. Необходима для удобства проведения тестов, например, возможность удалить 
сохраненный файл с Заметками, или сброс настроек приложения, вспомогательный функционал, которого нет в релизе
Основа для схемы Release. 
Параметры теста:
  Preprocessor Macros:
    Test: TEST=1

Также создана схема Test с конфигурацией для Release = Test
